# 游戏说明

本压缩包包含两个小游戏：

- `flying_bird.py`：Flying Bird，小鸟飞行躲避管道游戏
- `arrow_exit_puzzle.py`：Arrow Exit Puzzle，箭头逃脱解谜游戏
- `install_pygame_once.bat`：Windows 电脑首次运行 Python 版前，用于安装 pygame

如果你已经拿到 `.exe` 版本，可以直接双击 exe 运行，不需要安装 Python 或 pygame。

---

## 运行 Python 版前的准备

这两个 Python 版游戏都使用 `pygame` 制作。

如果是在 Windows 电脑上第一次运行 Python 版，请先双击：

```text
install_pygame_once.bat
```

安装完成后，再运行对应的 Python 文件：

```text
flying_bird.py
arrow_exit_puzzle.py
```

如果电脑没有安装 Python，需要先安装 Python 3。

---

## Flying Bird 玩法介绍

`Flying Bird` 是一个类似 Flappy Bird 的横版躲避游戏。

### 基本目标

控制小鸟不断穿过管道之间的空隙，尽可能获得更高分数。

### 操作方式

- 按空格键、鼠标点击或触屏点击：让小鸟向上飞一下
- `P`：暂停 / 继续
- `R`：游戏结束后重新开始
- 左上角“重置进度”：清空最高分记录

### 游戏规则

小鸟会持续下落，玩家需要不断点击让它保持高度。

每成功穿过一组管道，分数加 1。

如果撞到管道、地面，或者掉出安全区域，游戏结束。

游戏会自动保存最高分。

## Arrow Exit Puzzle 玩法介绍

`Arrow Exit Puzzle` 是一个箭头逃脱类解谜游戏。

### 基本目标

棋盘上有很多弯曲箭头。玩家需要按照正确顺序点击箭头，让所有箭头从棋盘中逃出。

### 核心规则

每个箭头都有一个箭头头部。

点击箭头后，系统只判断这个箭头“头部朝向的前方”是否被其他箭头挡住：

- 如果头部前方没有阻挡，箭头会沿着自己的弯曲路径钻出棋盘并消失。
- 如果头部前方被其他箭头挡住，本次点击失败，会扣除一颗心。

箭头自己的身体不会挡住自己，但会挡住其他箭头。

### 操作方式

- 鼠标点击 / 触屏点击：选择箭头
- 鼠标滚轮：缩放棋盘
- `P`：暂停 / 继续
- `R`：重新开始当前关卡
- `ESC`：返回或关闭弹窗
- 左上角“重置进度”：清空关卡模式进度，恢复到只解锁第 1 关

### 关卡模式

关卡模式一共有 100 个内置关卡。

默认只解锁第 1 关，通关后会逐步解锁后面的关卡。

胜利后可以点击“下一关”直接进入下一关。

### 随机模式

随机模式会生成一个新的随机关卡。

每个随机关卡都会显示一个 Seed，复制 Seed 后可以分享给别人，别人输入同一个 Seed 就能玩到同一张地图。

随机模式通关后：

- “再来一次”：使用相同 Seed 重新开始
- “来个新的”：重新生成一个新关卡

### Seed 模式

Seed 模式可以输入别人分享的 Seed。

如果 Seed 合法，并且关卡可解，游戏会加载对应关卡。

如果 Seed 错误、损坏或无解，会提示重新输入。

### 技能说明

游戏中有三个辅助功能：

- 网格：显示或隐藏棋盘辅助线
- 提示：高亮一个当前可以正确移出的箭头
- 擦除：直接删除一个箭头

普通模式下，每关的提示和擦除次数有限。

### 作弊模式

主界面右上角有 Admin 入口。

输入：

```text
admin
```

即可临时开启作弊模式。

作弊模式效果：

- 解锁全部内置关卡
- 取消时间限制
- 提示次数无限
- 擦除次数无限

作弊模式不会保存到本地。关闭游戏或刷新页面后，需要重新输入 `admin` 才能再次开启。

---

## 备注

Python 版和 HTML 版的 Arrow Exit Puzzle 使用同一套 100 个内置关卡 Seed，因此同一关卡在不同版本中内容一致。
